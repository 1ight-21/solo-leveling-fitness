# Solo Leveling Fitness

This project is a fitness application based on the themes of the "Solo Leveling" series. It aims to provide users with personalized fitness plans and progress tracking.

## Features
- User registrations with authentication
- Customized workout plans
- Progress tracking and analytics

## Technologies Used
- React Native for mobile application
- Firebase for backend and authentication

## Getting Started
1. Clone the repository: `git clone https://github.com/1ight-21/solo-leveling-fitness`
2. Navigate into the project directory: `cd solo-leveling-fitness`
3. Install dependencies: `npm install`
4. Create a `.env` file using the `.env.example` file as a template and set up your Firebase configuration.
5. Run the project: `npm start`
